Star Pyramid

Given an integer n, print a pyramid full of stars with height equal to n.

Input Format

The first line of input contains the integer n.

Output Format

The output should contain a pyramid filled with stars with height equal to n.

Example 1

Input

3

Output:

```java
    *
   * *
  * * *
```

Input

7

```java
        *

      * * *
     * * * *
    * * * * *
   * * * * * *
  * * * * * * *
```
